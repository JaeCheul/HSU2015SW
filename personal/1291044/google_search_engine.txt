구글 검색이 뛰어난 이유



1. 기술적 차이  

  - 페이지 순위(PageRank) 시스템의 알고리즘으로 객관적인 웹 페이지 순위를 계산한다.  
  이 시스템은 투표와 같이 페이지의 링크가 많을수록 랭크가 높아지고, 링크가 적으면 낮게 책정된다.
  랭크가 높은 페이지는 '중요도'가 높게 평가되어 검색 시 관련성이 높은 결과를 보여준다.  

  - 검색어에 해당하는 페이지의 제목이나 키워드만 보여주지 않는다.   
  URL 내용을 페이지에서 직접 불러오므로, 미리 보고 필요한 페이지만 골라 볼 수 있다.  


2. 사용자의 편의  

  - 다른 웹 페이지에서 검색할 경우, 사용자가 원하는 정보 뿐만 아니라 불필요한 광고들도 제공한다.  
  구글은 검색어에 모두 일치하는 결과만 보여주므로 관련 없는 검색 결과를 걱정할 필요가 없다.  

  - 사용자가 서버 오류로 인해 접속을 못할 경우를 대비해 웹 문서들을 저장한다.   
  저장된 페이지는 재검색이 필요 없으므로 원래 사이트보다 빠르게 확인할 수 있다.  


3. 지속적인 노력  

  - 매년 CSI(Crazy Search Ideas) 회의를 통해 검색 엔진을 지속적으로 개선을 한다.  
  이 회의는 개선을 위한 창조적인 아이디어를 모집하고 토론을 하는 회의이다.

  - 검색을 하는 사용자들의 질문 패턴을 파악하고, 사전에 예측하여 먼저 답을 주도록 한다.
  특히 구글은 15주년 기념일날 새로운 검색 알고리즘인 '허밍버드(Hummingbird)'를 발표하였다.
  이 알고리즘은 단순한 단어가 아닌, 일상의 대화체로 검색을 해도 질문의 문맥과 의도를 파악한다.
  이를 통해 사용자에게 보다 나은 검색결과를 제공할 수 있다.

  - 음성 검색, 이미지 검색 등 다양한 검색 방법을 제공하며 그 범위를 넓히고 있다.
  특히 이미지 검색은 URL, 업로드, 드래그 앤 드롭의 3가지 방법으로 원하는 사진을 올릴 수 있다.


4. 다양한 검색 기능  

  - 큰따옴표("")를 통해 해당 단어가 꼭 포함된 겸색 결과만 보여준다.  

  - 마이너스기호(-)를 통해 특정 단어만 제외된 검색 결과를 보여준다.  

  - site:를 통해 특정 사이트의 자료/정보만 검색하여 보여준다.
