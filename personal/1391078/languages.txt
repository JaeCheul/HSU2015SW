Python
1. Python은 우리의 언어와 가장 비슷한 언어입니다. 이런 점에서 Python을 공부하는 사람들 또한 쉽게 배울 수 있어서 전문가가 아니어도 개발을 쉽게 할 수 있습니다..
2. 거의 모든 환경에서 프로그램 개발이 가능합니다.  Python에서는 다른 언어를 포함 할 수 있는데, 이렇게 해서 빠른 속도를 필요로 하는 부분은 C로 개발하고 전체적인 틀을 Python으로 하기도 합니다.
3. 다양한 라이브러리을 제공해서 사용자가 조금 더 편하게 개발할 수 있도록 합니다.
4. Python을 실행하기 위해서는 코드의 정해진 줄을 잘 맞춰야 하는데, 이런 점들로 가독성이 좋아져 내가 만든 코드를 다른 사람이 볼 때에도 불편함이 없습니다.
5. 메모리가 자동으로 관리됩니다. 쓰레기 수집이라는 기능으로 필요할 때 메모리 관리를 모두 자동으로 합니다.

Ruby
1. 배우기 쉽습니다. 루비를 만든 사람이 루비를 만들 때부터 배우기 쉬운 언어 지향적으로 개발을 했다는 말이 나올 만큼 쉽습니다. 
2. 코드가 간결합니다. 예를 들어, JAVA로 개발 할 경우 길어질 코드가 Ruby on rails를 이용하면 훨씬 짧게 줄일 수 있습니다.
3. 순수 객체 지향적 언어라는 것입니다. 모든 것이 객체로 사용되어 문법이 어렵지 않습니다.
4. 자유로운 문법으로 개발하는 사람들에게는 직관적으로 프로그래밍을 할 수 있어서 편리하며 가독성이 좋습니다.
5. 문법이 자유로운 편이기 때문에 개발자는 자신이 원하는 데로 재미있게 프로그래밍이 가능합니다.

PHP
1. PHP는 개인 홈페이지 제작을 목적으로 한 언어에서 발달한 케이스입니다. 그렇기 때문에 사이트 제작 시에 유용한 함수가 잘 되어있습니다.
2. 문법이 간단하게 표현되어 코딩하게 되어 있고 직관적인 프로그래밍이 가능하여 편리하고,  장기적으로 프로젝트를 할 때 빠르게 개발이 가능합니다. 
3. Python이나 Ruby와 같이 변수에 타입이 따로 없는 언어라서 확장이 자유롭습니다. 또, 스크립트 언어라서 HTML 내부에 포함 시킬 수 있다.
4. Mysql과의 호환성이 좋습니다. sql문을 쓸 때 간단하게 쓸 수 있고, cmd 없이도 직접 데이터베이스를 구동할 수 있습니다.
5. 객체 지향 프로그래밍이 가능하여 조금 더 복잡한 프로그래밍을 할 수 있습니다.

Javascript
1. Javascript를 실행시키기 위해서 텍스트 문서와 같은 곳에서 바로 개발 할 수 있고 웹브라우저에서 실행되므로 개발 속도가 빠릅니다.
2.  Javascript는 HTML코드 내에서 바로 실행됩니다.  운영체제에 상관없이 개발이 가능하여 특정 작업 환경의 구애를 받지 않습니다.
3. Java를 기반으로 한 언어이므로 Javascript는 다른 언어에 비해 문법이 쉽고 Java를 배웠던 사람이라면 빠르게 프로그램을 만들 수 있습니다.
4. 다른 웹 페이지를 열거나 서버 응답 시간을 기다리지 않고 동적인 기능을 수행 할 수 있습니다.
5. 웹브라우저 안에서 실행되기 때문에 별도의 메모리관리가 따로 필요하지 않습니다.
