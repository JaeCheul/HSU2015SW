Python
파이썬의 장점
1. 쉽다
2. 금방 배운다
3. 메모리 소모가 적다
4. 같은 시간동안 더 많은 코딩이 가능하다
5. 개발 속도가 빠르다

파이썬은 개발기간단축을 중심으로 만들어진 언어이다. 그래서 파이썬의 최대 강점은
쉽고 금방 배운다는 것이다.초보자 분들에게는 컴퓨터 언어에 대해 쉽게 접하고 
빠르게 배울수 있게 해주고, 전문가 분들에게는 빠른 시간안에 개발을 할수있게 
개발 속도를 단축시켜준다. 파이썬은 다른 언어에 비해 코드의 라인 수도 매우 적으며
메모리 소모도 다른 언어에 비해 적다. 파이썬은 인터프린터 언어로 컴파일 언어와는 다르게
컴파일 과정이 생략되고 실행 중에 바로 한줄 한줄 번역해 가면서 실행된다. 속도가 떨어지는
단점이 있지만, 한줄 한줄 해석되는 언어이다보니 수정과 보완에 매우 좋다. 


Ruby
루비의 장점
1. 심플한 문법
2. 많은 OS에서 작동
3. 개발기간 짧음
4. 열외처리가능
5. 모든 데이터형식이 객체

루비는 일본인이 개발한 동적 객체 지향 스크립트 프로그래밍 언어이다. 루비는 객체 지향 언어로
모든 데이터 형식이 객체로 되어있다. 루비는 자바나 c언어에 비해 작은 분량의 프로그램으로 
컴퓨터에 동일한 명령을 하는것이 가능하고,매우 심플한 문법을 갖고 있어, 이해하기가 매우 
쉽고, 간결하다는 특징을 갖고 있다. 그 이유 때문에 개발기간이 매우 짧다. 개발기간이
절반에서 1/5까지 단축된다고 한다. 루비는 범용 프로그래밍 언어이다.웹 개발을 할수도 있고,
다양한 산업기기, 가전제품에도 사용된다. 웹 개발을 할때 객체지향적이여서 PHP보다 소스 코드
관리가 더 쉽다. 

PHP
PHP의 장점
1. 제반 환경 구축비용 무료
2. 모든 OS에서 작동
3. 데이터베이스와 연결작업 쉬움
4. 빠르게 코딩가능
5. 실행 속도가 빠름

PHP는 MySQL 이나 자바스크립트 등과의 연동이 매우 뛰어나다는 특징이 제일 큰 강점이라고 할 수 있다.
PHP는 인터프린터 언어로 컴파일 언어와는 다르게 컴파일 과정이 생략되고 실행 중에 바로 한줄 한줄
번역해 가면서 실행된다. 속도가 떨어지는 단점이 있지만, 한줄 한줄 해석되는 언어이다보니 
수정과 보완에 매우 좋다. PHP는 쉽고 문법이 간단하여 빠르게 코딩이 가능하고, c나 c++와 언어의
문법이 비슷하고 클래스를 지원하여 효율적인 코딩이 가능하다. PHP는 많은 프로그래머들에게 
쉽게 접근할 수 있다는 점이 있다. 자바스크립트와 같은 스크립트언어지만 보안에 매우 취약한점이
있다. 그리고 대형 프로젝트에 적합하지 않다는점이 있고, 자바스크립트만큼 역동적인 홈페이지를 
구현하기 힘들다.

Javascript
자바스크립트의 장점
1. 웹개발에 강함
2. 컴파일이 필요없어 개발이 빠름
3. OS에 상관없이 작동함
4. 웹서버에 부담이 적어 성능 향상
5. 수정 보완하기 좋음

자바스크립트는 동적인 홈페이지를 구현해내기 위해 개발된 언어이다. 자바스크립트는 자바와 
기본적인 문법이 같지만 같은 언어는 아니다. 자바스크립트는 정적인 PHP가 즉각적인 응답을 
받을 수 없는 부분들을 해결했다. 자바스크립트는 다른 웹개발 언어에 비해 웹서버에 주는 부담이
적다. 부담이 적어서 더 성능들을 향상 시킬수있다. 자바스크립트도 인터프리터 언어로 컴파일 
언어와는 다르게 컴파일 과정이 생략되고 실행 중에 바로 한줄 한줄 번역해 가면서 실행된다.
 속도가 떨어지는 단점이 있지만, 한줄 한줄 해석되는 언어이다보니 수정과 보완에 매우 좋다.
