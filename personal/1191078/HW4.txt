Phython

다른 언어들과 다른 파이썬의 가장 큰 특징은 end나 {}를 사용하지 않고 범위를 들여쓰기로 지정한다는 
특징을 갖고 있었습니다. 들여쓰기를 통해서 범위를 지정하다 보니까 기존의 언어에 익숙했던 저로서는 
불편 했지만 가끔 들여쓰기를 안하는 프로그래머의 코드를 볼 때 불편 했던 점이 있었는데 파이썬은 구
조적으로 이렇게 만들어서 가독성이 더 뛰어난 것 같습니다. 그리고 변수를 선언 할 때 따로 변수타입
을 지정 안해줘도 알아서 판단을 해주고 소스들이 다 공개 된 특징을 갖고 있어서 Phython을 사용하는 
사람들에게 편리하도록 해주는 특징이 있습니다.

Ruby

아무나 배우기 쉽도록 인터프리팅 방법을 사용 한다는 장점이 있고 타입명을 선언하지 않고 전역변수, 
지역변수, 인터스턴스변수인지만을 판별하면 됩니다. Ruby는 매우 객체 지향적인 언어이며 정수나 문자
열 등의 모든 데이터 형식은 객체로서 동작 하고 루비의 어떤 곳이던 개발자가 자기 마음대로 수정하거
나 삭제 할 수 있습니다. 이미 제공 되는 연산자나 코드를 수정하거나 삭제 할 수 있습니다. 그리고 
Phython은 들여쓰기로 범위를 지정한다면 Ruby는 end라는 명령어로 범위를 지정해줍니다.

PHP

PHP는 확장성이 뛰어나며 DataBase에 접근 할 때 접근성이 매우 뛰어난 언어 입니다. 가장 많은 사용자
를 보유 하고 있으며 오픈 소스 스크립트 언어로 잘 알려져 있습니다. 언어 자체가 가벼워서 실행 속도
가 굉장이 빠르다는 장점이 있고 소스를 가져오는 파싱이나 정규식을 사용학 매우 쉽습니다. 그렇지만 
PHP가 컴퓨터 또는 서버에 설치가 되있어야 실행이 가능하다는 단점이 있습니다.

Javascript

Javascript는 PHP와는 다르게 서버가 없어도 메모장과 브라우저만 있으면 프로그래밍하고 출력 화면을 
확인 할 수 있습니다. 브라우저만 있으면 출력을 볼 수 있는 이유는 플랫폼에 의존하지 않고 동작 하기 
때문입니다. 재사용이 가능하다는 객체 지향의 특징을 갖고 있으며 개인 PC에서 작동 하기 때문에 서버
에 부담을 적게 준다는 장점도 있습니다. HTML 페이지에 Java의 코드로 기능을 추가 시켜주는데 주로 
사용 되기도 합니다. 소스 코드가 공개 되기 때문에 보안이 취약 할 수 있습니다.
