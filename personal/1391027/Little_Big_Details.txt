Evernote : 자동으로 열 너비에 따라 날짜 형식을 변환시킨다. 기존에 열 너비에 따라 정보가 가려지는 것을 방지 하고 
쉽게 날짜를 파악할 수 있어 좋은 것 같다.

Yahoo Weather : 아주 추운 날씨에 화면 모서리에 서리가 낀 것을 보여준다. 시각적인 효과를 더해 날씨를 파악하는 데 
도움이 될 것 같다.

Chrome : 인터넷 연결이 끊키면 게임으로 전환시킨다. 인터넷 연결을 다시 기다리는 동안 지루하지 않게 재미를 더해준다. 
인터넷이 끊긴 답답함을 잠시나마 달랠 수 있는 아이디어인 것 같다.

OnSite : 이메일을 통해 가용성 업데이트가 성공하면 춤추는 남자화면이 나타난다. 성공 표시를 단순하게 하지 않고 동적인 
이미지를 추가하여 재미를 더한 것 같다.

NY Times : 태블릿 장치에서 두 번 누르면 전체 화면을 확대하는 것이 아니라 텍스트의 크기만 확대한다. 기존에 화면을 확대
하면 이미지도 같이 확대되어 핵심 정보를 놓치게 되는데 텍스트만 확대하여 원하는 정보만을 쉽게 찾아낼 수 있을 것 같다.

Google Maps : 하루의 시간에 따라 정확하게 그림자 표시를 구축하였다.시간에 따라 그림자를 볼 수 있어 신기하고  구글 맵을 
이용해 다양한 곳에 응용이 될 수 있을 것 같다.

ReadMe : 패스워드를 입력할때 올빼미 그림이 눈을 가린다. 간단한 동작이지만 로그인 하는동안 재밌는 요소를 더했다. 
무의식중에 로그인을 하더라도 눈길을 한번 더 사로잡는 아이디어라고 생각한다.

Amazon : 책의 뒷면, 내부 등 책의 세부정보를 3D애니메이션을 통해 확인이 가능하다. 책의 겉면만 보여주는 기존의 방식과
달리 애니메이션을 통해 보여주는 것이 효율적이다.

Foursquare : 생일 때 프로그램을 열면 생일축하한다는 인사말이 뜬다. 사용자의 생일에 맞게 이런 이벤트를 해주는게 좋은 것 같다.

Basdcamp : 양식 필드에 오류가 생기면 그림이 놀라는 표시가 된다. 이것또한 시각적인 요소를 가미하여 사소하지만 흥미를 
주는 요소이다.

Waymate : 여행을 검색하면 여행지 도착일부터 삼일간의 날씨 정보를 보여준다. 날씨 정보를 미리 확인하여 날씨에 대한 대비를 
할 수 있어서 좋다.
