 자바 스크립트는 웹 서버 대신 컴퓨터 프로세서에서 실행됩니다. 따라서 인터넷 속도나 웹서버의 부담을 줄여줍니다.
 자바 스크립트는 상대적으로 영어와 비슷한 쉬운 문법들 그리고 기존 프로그래밍 언어인 C, Java와 비슷한 문법이라 배우기 쉽습니다. 
 파이어 폭스의 그리스몽키 같은 서드 파티 애드온들이 개발자들의 부담을 덜어주되 더욱 훌륭한 기능들을 수행하는 웹페이지를 만들어 줍니다.
 자바 스크립트가 웹에서 수행하는 작업들은 보통 단순해서 메모리를 크게 잡아먹지 않습니다.
 운영체제에 관계없이 자바스크립트는 쌩쌩하게 돌아갑니다.

php : http://i.imgur.com/IzFFRvf.png
 
 PHP는 HTML에 <php? ?>로 쉽고 분명하게 스크립트를 끼워넣을 수 있습니다. 
 사실상 PHP는 리눅스 뿐만이 아니라 다른, 예를 들어 윈도우, 유닉스, IBM 운영체제들에서도 작동합니다.
 개발중에도 PHP의 유연성 덕분에 개발자들은 본인들의 소중한 시간을 아끼며 코드를 마음껏 변경할 수 있습니다.
 PHP의 인기덕에 수없이 많은 무료 소스코드들이 넘쳐나고 있습니다. 오픈 소스 덕분이죠.
 많은 호스팅 회사들이 PHP를 돌릴 서버를 아주 적은 비용으로 마련해주어 여러분들은 정말 손쉽게 웹사이트를 제작 할 수 있습니다.

ruby : http://i.imgur.com/drJgqY5.png

 루비는 객체 지향 프로그램을 지향하고 있습니다. 객체를 이용해 자료 구조 손쉽게 다룰 수 있죠.
 루비는 기본적인 프로그래밍 언어들에 대한 지식만 있으면 정말 빠르게 습득할 수 있습니다. 
 루비의 블록(일회성 익명 함수)구문은 파이선의 람다와 다르게 괄호만 잘 묶어준다면 여러줄에 걸쳐 코딩을 할 수 있습니다.
 루비는 인터프리터 언어로서 루비를 작성하고 실행하는데 컴파일러가 필요하지 않습니다.
 루비의 믹스-인은 불러온 클래스 혹은 모듈의 기능을 확장하거나 유저 코드에 손쉽게 오버로딩될 수 있도록 만들어 줍니다. 

python : http://i.imgur.com/dVzq5kn.png

 파이선의 문법은 읽기쉽고 깔끔합니다. 몇가지 규칙(들여쓰기, 중괄호와 세미콜론 제거)를 도입하여 프로그래머들에게 가독성이 높은 환경을 제공합니다.
 파이선은 자바스크립트보다 보다 거대한 프로그램과 코드 재사용성을 가지고 있습니다. 객체지향 프로그래밍을 덕분에 말이죠.
 파이선은 다양한 운영체제에서 돌아갈 뿐만 아니라 심지어 다른 플렛폼에서도 비슷한 인터페이스를 가지고 있습니다.
 파이선은 의사 코드(인간이 알아듣기 쉬운 코드)와 닮아 꽤나 배우기 간단합니다.
 파이선의 커뮤니티가 막대한 분량의 지원을 유저들에게 제공하여 개발환경이 좋습니다.
